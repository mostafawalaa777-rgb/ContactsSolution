# ContactsSolution
# Contacts Management System

## Overview

A Contacts Management System developed using C#, SQL Server, and ADO.NET following 
the Three-Layer Architecture pattern (Presentation Layer, Business Layer, and Data Access Layer).

## Features

### Contact Management

* Add new contacts
* Update existing contacts
* Delete contacts
* Find contacts by ID
* Check if a contact exists
* Display all contacts

### Country Management

* Add new countries
* Update country information
* Delete countries
* Find countries by ID or name
* Check if a country exists
* Display all countries

## Technologies Used

* C#
* SQL Server
* ADO.NET
* Three-Layer Architecture
* Object-Oriented Programming (OOP)

## Project Structure

* Presentation Layer
* Business Layer
* Data Access Layer

## Database Operations

* CRUD Operations (Create, Read, Update, Delete)
* Parameterized SQL Queries
* SQL Server Database Integration
Learning Objectives:
- Applying Three-Layer Architecture.
- Working with SQL Server using ADO.NET.
- Building reusable Business Logic.
- Implementing CRUD operations.
